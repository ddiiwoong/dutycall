# 의무통화

은행·카드사·항공사 등 주요 고객센터 전화번호 모음 웹앱.
클릭 한 번으로 바로 전화 걸기, 월별 통화 체크 및 누적 시간 관리.

🔗 **https://dii.kr**

## 기능

- 고객센터 전화번호 리스트 (통화시간 내림차순 정렬)
- 전화 버튼 클릭 시 바로 전화 연결 + 자동 체크
- 복수 통화 카운트 (같은 곳에 여러 번 전화 가능)
- 이번 달 누적 통화시간 표시
- 매달 1일 자동 초기화 (통화 체크만 리셋)
- 연락처 추가/삭제 (편집 모드)
- 목록 초기화 (기본 목록으로 복원)
- 모바일 최적화 (`tel:` 링크)

## 기술 스택

- HTML / CSS / JavaScript (프레임워크 없음, 단일 파일)
- `localStorage` 기반 데이터 저장 (서버 불필요)
- GitHub Pages 호스팅

## 데이터 저장

모든 데이터는 브라우저 `localStorage`에 저장됩니다.

| 키 | 내용 | 초기화 시점 |
|---|---|---|
| `dutycall_done` | 통화 체크 횟수 | 매달 자동 / 초기화 버튼 |
| `dutycall_custom` | 사용자 추가 연락처 | 목록 초기화 버튼 |
| `dutycall_hidden` | 삭제한 연락처 | 목록 초기화 버튼 |
| `dutycall_month` | 마지막 접속 월 | - |
